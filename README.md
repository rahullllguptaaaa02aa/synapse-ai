# 🚀 Synapse AI — Premium AI Data Automation Platform

A modern, high-performance SaaS landing page built for the **Next-Gen AI Platform Speed Run** frontend engineering challenge.

Synapse AI is a fictional AI-powered data automation platform that helps teams build, orchestrate, and monitor intelligent data pipelines with enterprise-grade performance, responsive design, and optimized frontend architecture.

---

## ✨ Overview

Synapse AI demonstrates a production-style frontend architecture with a focus on:

* Premium SaaS UI/UX
* Responsive design
* Performance optimization
* Semantic HTML & SEO
* State isolation
* Clean React architecture
* Native CSS animations

The project showcases how a modern AI automation platform could present its product, features, pricing, and customer success stories through a polished landing page.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/synapse-ai.git
```

Navigate into the project

```bash
cd synapse-ai
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
